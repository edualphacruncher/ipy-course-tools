Install Guide
=============

.. note::

    It is best practice to install Python projects in a virtual environment,
    which can be created and activated as follows using Python 3.6+.

    .. code-block:: console

        $ python -m venv venv --prompt myvenv
        $ source venv/bin/activate
        (myvenv) $

The simplest way to install ipy-course-tools is using Pip.

.. code-block:: console

    $ pip install ipy_course_tools

This will install ``ipy_course_tools`` and all of its dependencies.

Report Bugs
===========

Report bugs at the `issue tracker <https://github.com/matek-alphacruncher/ipy_course_tools/issues>`_.

Please include:

  - Operating system name and version.
  - Any details about your local setup that might be helpful in troubleshooting.
  - Detailed steps to reproduce the bug.
